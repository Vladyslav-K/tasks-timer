# _Tasks timer_

## Track how much time you spend on tasks!

### [Link on Github Page](https://vladyslav-k.github.io/tasks-timer)

---

### Install dependencies:

```
$ npm install
```

### Build for production:

```
$ npm run build
```

### Build for development:

```
$ npm run start
```

### Run test

```
$ npm run test
```
